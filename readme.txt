== Description ==

You can use this plugin to include your "follow me" tabs on your WordPress site as icons on different screen positions. Clicking on the icon it will redirect your visitor to your social network page.
Features

- It supports 4 different tab styles that can be placed on the left or right side of your site with 2 different icons sizes
- You can add your Follow Me tabs as Widget with 3 different icon styles
- Shortcodes are also included for your page content

Supported Social Networks

- Facebook
- Google +1
- LinkedIn
- Twitter
- YouTube
- Blogger
- Skype
- MySpace
- Pinterest
- RSS Feed
- last.fm
- Deviantart

Available shortcodes:
[follow_me_icons type="zoom" size="small"]
[follow_me_icons type="zoom" size="large"]
[follow_me_icons type="color" size="small"]
[follow_me_icons type="color" size="large"]
[follow_me_icons type="grey" size="small"]
[follow_me_icons type="grey" size="large"

Plugin Demo:
http://follow.code4site.com 

Note: This plugin is not using any data from any 3rd pary site and is not sending any data to any 3rd party site. Everything inserted as data is saved in your WordPress database.

== Installation ==

1. Upload follow_me_tabs folder and all its files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings->Follow Me Tabs and set your social networks
4. You can use follow me tags as widget on Appearance->Widgets
5. You can add shortcodes in your page content
